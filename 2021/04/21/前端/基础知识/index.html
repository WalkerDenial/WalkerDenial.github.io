基础数据类型

  1. Number – 数字，双精度 64 位格式
  2. String – 字符串
  3. Boolean – 布尔
  4. Symbol – 符号
  5. null – 空
  6. undefined – 未定义
  7. Object – 对象
    1. Function – 函数
    2. Array – 数组
    3. Date – 日期
    4. RegExp – 正则表达式

Tip1: NaN + Number = NaN

Tip2: Infinity/-Infinity

Tip3: false、0、空字符串(“”)、NaN、null 和 undefined 被转换为 false

关键字说明

  1. let：仅限于块作用于，不可对同一名称二次声明；
  2. var：可以是全局作用于，可以对同一名称二次声明；
  3. const：let 的补充，相当于 final 类型，不可修改；
